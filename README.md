# FortiGate Monitor

Monitor de consumo de tráfico para FortiGate.

## Requisitos

- Rocky Linux 8/9
- Python 3.9+
- rsyslog
- SQLite

## Instalación

```bash
./install.sh
```

## Configuración

Editar:

config.yaml

## Iniciar

```bash
systemctl start fortigate-monitor
```

## Estado

```bash
systemctl status fortigate-monitor
```
